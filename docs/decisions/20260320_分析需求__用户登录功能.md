## 1. MODELING (逻辑预演)

### 实体

| 实体 | 核心属性 | 职责 |
|------|----------|------|
| **User** | id, username, email, password_hash, status, created_at | 用户主体，存储身份信息 |
| **OAuthConnection** | id, user_id, provider, provider_user_id, access_token | 第三方登录绑定关系 |
| **Session** | id, user_id, access_token, refresh_token, expires_at | 会话管理，支持多端登录 |
| **LoginLog** | id, user_id, ip, user_agent, status, created_at | 审计追踪 |

### 状态

```
用户生命周期：
[Visitor] --注册--> [Pending] --验证邮箱--> [Active] --封禁--> [Suspended]
                              ↓
                          [Deleted] (软删除)

登录流程状态：
[Init] --提交凭证--> [Validating] --成功--> [Success]
                      ↓
                   --失败--> [Failed] (记录失败次数)
                      ↓
                   --超过阈值--> [Locked] (账户临时锁定)
```

### 边界

| 层级 | 后端职责 | 前端职责 |
|------|----------|----------|
| **认证** | 密码校验、Token 签发、OAuth 流程 | 表单收集、Token 存储 |
| **授权** | 权限校验、路由鉴权 | 路由守卫、UI 权限控制 |
| **会话** | Token 刷新、会话注销 | Token 自动续期、登出清理 |
| **审计** | 登录日志记录 | 错误信息展示 |

## 2. SCHEMA (数据驱动)

### 用户表

```sql
CREATE TABLE users (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    username        VARCHAR(50) UNIQUE NOT NULL,
    email           VARCHAR(255) UNIQUE NOT NULL,
    password_hash   VARCHAR(255),  -- NULL 表示仅第三方登录
    nickname        VARCHAR(100),
    avatar_url      VARCHAR(500),
    status          VARCHAR(20) DEFAULT 'pending',  -- pending/active/suspended/deleted
    email_verified  BOOLEAN DEFAULT FALSE,
    login_fail_count INTEGER DEFAULT 0,
    locked_until    TIMESTAMP,
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_users_email ON users(email);
CREATE INDEX idx_users_username ON users(username);
CREATE INDEX idx_users_status ON users(status);
```

### OAuth 绑定表 (oauth_connections)

```sql
CREATE TABLE oauth_connections (
    id               UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id          UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    provider         VARCHAR(20) NOT NULL,  -- wechat/github/google
    provider_user_id VARCHAR(255) NOT NULL,
    access_token     TEXT,
    refresh_token    TEXT,
    expires_at       TIMESTAMP,
    created_at       TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    
    UNIQUE(provider, provider_user_id)
);

CREATE INDEX idx_oauth_user ON oauth_connections(user_id);
```

### 会话表

```sql
CREATE TABLE sessions (
    id            UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id       UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    access_token  VARCHAR(500) UNIQUE NOT NULL,
    refresh_token VARCHAR(500) UNIQUE,
    device_info   JSONB,
    ip_address    INET,
    expires_at    TIMESTAMP NOT NULL,
    created_at    TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_sessions_token ON sessions(access_token);
CREATE INDEX idx_sessions_user ON sessions(user_id);
```

### 登录日志表 (login_logs)

```sql
CREATE TABLE login_logs (
    id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id     UUID REFERENCES users(id),
    login_type  VARCHAR(20) NOT NULL,  -- password/oauth
    provider    VARCHAR(20),           -- wechat/github
    ip_address  INET,
    user_agent  TEXT,
    status      VARCHAR(20) NOT NULL,  -- success/failed
    fail_reason VARCHAR(255),
    created_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_login_logs_user ON login_logs(user_id);
CREATE INDEX idx_login_logs_created ON login_logs(created_at);
```

## 5. 风险评估

### 高风险

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| **密码存储安全** | 数据泄露导致用户凭证暴露 | 使用 bcrypt (cost≥12)，禁止明文存储 |
| **JWT 泄露** | 账户被盗用 | Token 有效期短 (2h)，refresh_token 单次使用，支持主动吊销 |
| **OAuth 回调伪造** | 账户接管 | 严格校验 state 参数，使用 PKCE |

### 中风险

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| **暴力破解** | 账户被撞库 | 登录失败锁定 (5次/15分钟)，IP 限流 |
| **第三方服务不可用** | OAuth 登录失败 | 降级提示，记录日志，监控告警 |
| **邮箱验证邮件延迟** | 用户体验差 | 异步发送，提供重发按钮 |

### 技术债务预警

1. **微信 OAuth 需要企业资质**：个人开发者无法申请网页授权，建议初期仅支持 GitHub
2. **Token 吊销机制**：当前设计依赖 Redis 黑名单，如无 Redis 需改用数据库方案
3. **多端登录策略**：当前未限制多设备登录，后续可能需要"踢出其他设备"功能
