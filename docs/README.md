# 项目文档目录

此目录存放项目的所有文档资产。

## 目录结构

```
docs/
├── decisions/     # 架构决策记录 (ADR)
├── api/           # API 规格文档
├── schema/        # 数据模型文档
├── backend/       # 后端开发日志
├── frontend/      # 前端开发日志
└── testing/       # 测试文档
```

## 文档规范

### 架构决策记录 (ADR)
- 路径：`docs/decisions/`
- 命名：`YYYYMMDD_topic.md`

### API 契约
- 路径：`docs/api/`
- 命名：`{module}.md`

### 数据模型
- 路径：`docs/schema/`
- 命名：`{entity}.md`
